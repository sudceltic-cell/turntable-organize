# Turntable Release Organizer

A private, client‑side music release manager for rappers, producers, and artists.  
Built with vanilla HTML, CSS, and JavaScript – no frameworks, no backend, no data leaves your browser.

## Features
- 🎚 Animated turntable release type picker
- 🖼 Cover art upload with reattach support
- 🎵 Track list with waveform audio players (Web Audio API + Canvas)
- ✍️ Lyrics panel, mood tagging, progress tracker (Lyrics → Recorded → Mixed → Mastered)
- 👥 Collaborator & producer credits, freestyle/remix toggles, session notes
- 🎬 Video clip lightbox
- ↕️ Drag‑and‑drop reordering + arrow buttons
- ▶️ Play All sequential playback
- ⌨️ Keyboard shortcuts: `N` to add track, `Space` to play/pause, `Esc` to close
- 🖨 Export release sheet (printable)
- 💾 Auto‑save to localStorage, undo delete
- 📲 Installable Progressive Web App (PWA) – works offline
